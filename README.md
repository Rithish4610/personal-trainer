# Personal Trainer–Style Fitness Tracking Website

A rule-based fitness tracker that acts like a disciplined personal trainer. No AI, no external APIs—just logic, nutrition rules, and consistency tracking.

## Features
- User authentication (username, password, goal)
- Daily food logging (time, food, quantity)
- Nutrition engine (food database, calculates protein/carbs/fiber)
- Daily nutrition summary with color indicators
- Rule-based trainer feedback
- Night review system with daily score
- Consistency and streak tracking

## Tech Stack
- Backend: Node.js (Express)
- Frontend: React
- Database: SQLite

## Getting Started
1. Install dependencies in both `backend` and `frontend` folders.
2. Start the backend server.
3. Start the frontend app.

---

This project is a demonstration of rule-based logic for fitness tracking, with no AI or external services.
